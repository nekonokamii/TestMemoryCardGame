Le Memory Game est un jeu de cartes où le but est de retrouver des paires de cartes identiques.
Le jeu se joue sur une grille de 4x3 cases, chacune contenant une image.
Le but est de retourner toutes les cartes en trouvant les paires. Le jeu se termine lorsque toutes les cartes sont retournées.
